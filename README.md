## R4G - WebApplication

Front-end: VueJS <br/>
Back-end: Laravel

### INSTALLATION
[installation yarn (global) => npm install -g yarn] <br/>
VueJS => npm install --save / yarn add <br/>
Laravel => composer update <br/>

### RUNNING
VueJS => npm run serve / yarn serve <br/> 
Laravel => php artisan serve <br/>
